📡 Endpoints Principales

  • GET /api/ - Información de la API
  • POST /api/register - Registrar usuario
  • POST /api/login - Iniciar sesión
  • GET /api/challenges - Listar challenges
  • POST /api/submit - Enviar solución
  • GET /api/leaderboard - Ver ranking

💻 Cliente Python

Usa el cliente oficial para interactuar con la API:

from grader_qiskit_client import *

# Registrar usuario
register('username', 'email@example.com', 'password')

# Ver challenges
show_challenges()

# Enviar solución
submit(1, "tu código aquí")

# Ver leaderboard
show_leaderboard()
